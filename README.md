# HHRaiser

**HHRaiser** — это сервис на Go, который автоматизирует процесс поднятия резюме на платформе [hh.ru](https://hh.ru). Управление осуществляется через Телеграм-бота.

## 🚀 Основные возможности

- **Автоматический подъем резюме**: Настройка расписания.
- **Управление через Telegram**: Простые команды для старта, остановки и проверки статуса.
- **Статус выполнения**: Отчет об успешном подъеме или ошибках прямо в чат.

## 📦 Стек технологий

- **Язык**: Go
- **Telegram API**: [go-telegram-bot-api](https://github.com/go-telegram-bot-api/telegram-bot-api)
